# COGAR_assignment
Subgroup B1: UR5e + 2FG7 Integration
Assignment B1a: UR5e Setup, Motion Planning, and 2FG7 Force-Grasp Validation (REAL-ROBOT)
What to do: Set up and validate the UR5e robotic arm with ROS2 Control and MoveIt2 for real-robot motion planning and execution, define the cabinet and surrounding environment for collision avoidance, integrate the 2FG7 gripper in ROS2, and test force-based grasping on simple objects.
    1. Configure UR5e with ur_robot_driver, ROS2 Control, and MoveIt2 
    2. Verify joint-space and Cartesian motion on the real robot 
    3. Define the cabinet and nearby environment as collision objects in the planning scene 
    4. Configure collision avoidance constraints for safe robot motion 
    5. Test planning pipelines, collision checking, and trajectory execution reliability 
    6. Set up communication and control of the 2FG7 gripper in ROS2 
    7. Access and interpret force-related feedback from the 2FG7 gripper 
    8. Implement and test force-based grasp control on simple objects 
    9. Evaluate grasp stability and repeatability under different grasp conditions 
Software needed: ur_robot_driver, ROS2 Control, MoveIt2, OnRobot drivers, ROS2 Humble
Research needed: UR5e real-robot setup, MoveIt2 configuration, planning scene and collision modeling, 2FG7 communication interface, force-based grasping
Deliverables: Working UR5e setup with MoveIt2, cabinet-aware collision avoidance, integrated 2FG7 gripper control, and validated force-based grasping on simple objects
